# Account Security

## Password reset
- From the sign-in page, choose **Forgot password** and enter your account email.
- A password-reset link is emailed; the link expires in **30 minutes**.
- Reset links are single-use.

## Multi-factor authentication (MFA)
- We strongly recommend enabling MFA from **Settings → Security**.
- Supported factors: TOTP authenticator apps and FIDO2/WebAuthn security keys.
- Enterprise tenants can enforce MFA company-wide.

## Suspicious activity
- Sign-in from a new device or unusual location triggers an email alert.
- If you didn't initiate the sign-in:
  1. Change your password immediately.
  2. Revoke active sessions from **Settings → Security → Sessions**.
  3. Contact SupportMate to flag the account for review.

## Account lockout
- Five failed sign-in attempts trigger a temporary 15-minute lockout.
- A support specialist can release a lockout earlier after identity verification.

## What SupportMate will never do
- Ask for your password, MFA code, or full payment-card details.
- Send links via the chat that go to non-corporate domains.
